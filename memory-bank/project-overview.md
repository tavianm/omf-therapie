# Project Overview

**Last Updated:** January 14, 2025

## Project Name

OMF Therapie

## Description

Professional therapy practice website for Oriane Montabonnet, featuring service information, therapist credentials, blog content, and contact functionality. The site focuses on accessibility, user experience, and providing comprehensive information about therapeutic services.

## Primary Goals

- Present therapeutic services professionally and accessibly
- Provide educational blog content on mental health topics
- Enable easy contact and appointment scheduling
- Maintain high accessibility standards (WCAG compliance)
- Deliver fast, responsive user experience

## Target Audience

- Individuals seeking therapy services in France
- People interested in mental health and well-being topics
- Potential clients looking for information about therapeutic approaches
- Visitors seeking contact information and appointment booking

## Key Features

1. **Service Information** - Detailed presentation of therapy services offered
2. **Therapist Profile** - About section with qualifications and approach
3. **Blog System** - Articles on mental health, therapy, and well-being
4. **Contact Form** - EmailJS-powered contact functionality
5. **Accessibility Features** - Full WCAG compliance with regular audits
6. **Responsive Design** - Mobile-first approach with Tailwind CSS
7. **SEO Optimization** - Meta tags, sitemap generation, structured data
8. **Performance** - Lazy loading, code splitting, optimized assets
